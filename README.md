# SQL Data Cleaning & Exploratory Data Analysis

## 📊 Project Overview
This project focuses on cleaning and analyzing a global layoffs dataset using SQL.

## 🔧 Tools Used
- MySQL
- SQL

## 📌 Key Tasks
- Data Cleaning (handling nulls, duplicates)
- Exploratory Data Analysis
- Trend Analysis using SQL

## 📊 Key Insights
- Layoffs peaked in 2022, with tech companies contributing the highest layoffs
- Certain companies showed repeated layoffs across multiple years

## 🚀 Skills Demonstrated
- SQL (GROUP BY, JOINS, WINDOW FUNCTIONS)
- Data Analysis
- Problem Solving
